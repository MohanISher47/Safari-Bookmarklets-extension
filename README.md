# The Bookmarklets Extension by T.E.D.A.

An Extension made to make getting and using bookmarklets way easier

## How to add bookmarklets that you want to add
Go to https://github.com/MohanISher47/MohanISher47.github.io and make issue to add 
one or more bookmarklet be sure to include each of the bookmarklets' code and title.
Then, I will add the bookmarklet(s) later or sooner.
